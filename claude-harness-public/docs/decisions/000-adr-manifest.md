# Claude Harness — Architectural Decision Records

## Format

All ADRs follow [MADR 4.0](https://adr.github.io/madr/) format. Use `core/templates/adr-template.md` for new ADRs.

## Numbering

Claude-harness uses its own ADR numbering sequence (001, 002, ...), independent of downstream projects. This avoids collision when projects have their own ADR series.
